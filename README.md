## ğŸ“˜ Project Overview

The **Packet Sniffer Simulator** is a browser-based educational tool that simulates how real packet sniffers like **Wireshark** work.  
Instead of capturing live network packets (which requires admin permissions), it **generates synthetic packets** using JavaScript and displays them in real-time.

This project is ideal for:
- Computer Network Lab  
- B.Tech / MCA Mini Projects  
- Networking Learning  
- GitHub portfolio enhancement  
- Classroom demonstrations
